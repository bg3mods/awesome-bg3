## Mods

All the following mods should be unpacked with [LSLib](https://github.com/Norbyte/lslib) before you can check their files and learn from them.

- [Sample Magic Ring Mod](https://www.nexusmods.com/baldursgate3/mods/278) by AnteMaxx: Level 1 difficulty. A template mod for magic ring and spell modding. Contains 4 magic rings.
- [Sample Equipment Mod](https://www.nexusmods.com/baldursgate3/mods/249) by AnteMaxx: Level 2 difficulty. A template mod for Armor and Weapon modding, and for implementing custom icons. Contains 3 armors, 1 weapon and 1 backpack.
- [Basket Full of Equipment](https://www.nexusmods.com/baldursgate3/mods/87) by AnteMaxx: Level 3 difficulty. Over 700 pieces of new armor and equipment, mainly for humanlike races. Very useful to check for advanced functions, such as adding spells to equipment, etc.
- [Daughter of Lolth](https://www.nexusmods.com/baldursgate3/mods/1515) by Rukongai: Useful if you want to learn BG3 scripting and how to register listeners.

## Tools

- [BG3 MultiTool](https://github.com/ShinyHobo/BG3-Modders-Multitool) by ShinyHobo: Utility for quickly generating Baldur's Gate 3 mod packs from an unpacked workspace.
- [BG3 MiniTool](https://github.com/Padme4000/BG3-Mini-Tool) by Padme4000: Mini tool for editing some of the lsx files for baldurs gate 3 modding
- [BG3 Script Extender](https://github.com/Norbyte/bg3se) by Norbyte: required by many mods, and indispensable to access LUA script from within your mod or debugging them in the console
- [LSLib](https://github.com/Norbyte/lslib) by Norbyte: Tools for manipulating Divinity Original Sin and Baldur's Gate 3 files
